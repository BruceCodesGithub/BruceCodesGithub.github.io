YARSAW

Yet Another Random Stuff API Wrapper

YARSAW is an API Wrapper for the Random Stuff API. 

  1. YARSAW is an async wrapper for faster and better usage.
  2. This package wraps the whole API.
  3. Offers:
    • AI Response (chatbots)
    • Canvas to edit your images easily into various templates and memes
    • Covid-19 Stats
    • Jokes
    • Facts
    • Waifus
    • Weather Stats
    • Get Various types of images including memes, aww, dogs and cats, etc.
  4. Fast and async
  5. Updated constantly
